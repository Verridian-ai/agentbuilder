# Agent Builder GitHub MCP

A comprehensive Model Context Protocol (MCP) server providing 96 GitHub integration tools.

## Features

- Complete GitHub API coverage
- Repository management
- Issue and PR operations
- GitHub Actions integration
- Security scanning
- Analytics and insights

## Installation

```bash
cd agent-builder-github-mcp
pip install -r requirements.txt
./run.sh
```

## Available Tools

- 12 Repository Management tools
- 4 File Operations tools
- 5 Branch Management tools
- 9 Issue Management tools
- 9 Pull Request tools
- 6 GitHub Actions tools
- 6 Security tools
- And 45+ more...

## Usage

Start the MCP server:

```bash
./run.sh
```

The server will be available for MCP client connections.
